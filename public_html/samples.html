Полезные книги

Здесь приведу книги, которые могут пригодиться в не простой ситуации разработки и повышения квалификации

  • Antonio Goncalves - Beginning Java EE 7 Книга по Java EE технологиям. Servlets , beans, orm - всё здесь. Русское издание.
  • Joshua Bloch - Effective Java Книга по Java. Как, какой и когда писать код. Как и когда не писать. Русское издание.
  • Brian Goetz - Java Concurrency in Practice Книга по Java Concurrency API. Английское издание.
  • Craig Walls - Spring in Action Книга по фреймворку Spring. Русское издание.
  • Jim Adams - Programming Role Playing Games with DirectX Старая редкая книга по программированию на Directx 7. Содержит примеры кода. Пригодилась как-то раз для исправления бага однйо сторой игры на DirectPlay.Английское издание.
